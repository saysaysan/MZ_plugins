# PKRN_DQBattleStatus

RPGツクールMZ用のプラグインです。

バトル画面のステータスウィンドウを画面上部にドラクエ風レイアウトで表示します。

## 導入方法

ダウンロードした `PKRN_DQBattleStatus.js` を、プロジェクト内の `js/plugins` フォルダーの中に入れてください。

## 機能詳細

デフォルトのバトルステータスウィンドウ（画面右下・顔グラ付き）を廃止し、
画面上部に以下のようなレイアウトで表示します。

```
┌──────┬──────┬──────┬──────┐
│アリス │ボブ  │キャロ│デイブ│
│HP ██ │HP ██ │HP ██ │HP ██ │
│MP ██ │MP ██ │MP ██ │MP ██ │
│TP ██ │TP ██ │TP ██ │TP ██ │
│Lv 5  │Lv 8  │Lv 6  │Lv 7  │
└──────┴──────┴──────┴──────┘
```

- 各アクターが縦並び（名前 → HP → MP → TP → Lv）で表示されます
- TPはシステム設定の「TPの表示」がONのときのみ表示されます
- HPが危機的な状態の場合、名前が赤色で表示されます
- 最大5人まで横並びで表示できます

## プラグインパラメータ

| パラメータ | デフォルト | 説明 |
|---|---|---|
| 最大パーティ人数 | 4 | 戦闘に参加できる最大メンバー数（1〜5）。`Game_Party.maxBattleMembers` にも反映されます。 |
| ウィンドウ不透明度 | 255 | ウィンドウ枠と背景の不透明度（0: 完全透明 〜 255: 完全不透明）。 |

## 5人パーティについて

「最大パーティ人数」を5に設定すると、5人まで戦闘に参加できます。
5人目のメンバーを追加するには、イベントコマンドの「パーティの入れ替え」でメンバーを追加してください。

## PKRN_DQBattleLog との併用

`PKRN_DQBattleLog`（バトルログを画面下部に表示するプラグイン）と併用することで、
よりドラクエらしいバトル画面を実現できます。
