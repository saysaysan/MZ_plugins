# PKRN_DQBattleLog

RPGツクールMZ用のプラグインです。

バトルログウィンドウを画面上部から下部に移動し、ドラクエ風の表示にします。

## 導入方法

ダウンロードした `PKRN_DQBattleLog.js` を、プロジェクト内の `js/plugins` フォルダーの中に入れてください。

## 機能詳細

デフォルトのバトルログウィンドウは画面上部に表示され、枠もなく半透明な状態です。
このプラグインを導入すると、バトルログウィンドウが **パーティのステータスウィンドウのすぐ上** に移動し、
ウィンドウの枠や背景が表示されるようになります。

## プラグインパラメータ

| パラメータ | デフォルト | 説明 |
|---|---|---|
| 最大行数 | 3 | バトルログウィンドウに表示できる最大行数（1〜10）。 |
| ウィンドウ不透明度 | 255 | ウィンドウ枠と背景の不透明度（0: 完全透明 〜 255: 完全不透明）。 |
| 背景塗り不透明度 | 160 | テキスト背後の黒背景の不透明度（0: 透明 〜 255: 不透明）。 |
